# Raw files

## How write raw data
The data has to be inserted in a markdown file. It can have a title `# Title`. Each line (unless blank) is equivalent to a single versicle.

Each file is equivalent to a book chapter.

Example:

```markdown
# Mateu 1
βιβλος γενεσεως ιησου χριστου υιου δαυιδ υιου αβρααμ
αβρααμ εγεννησεν τον ισαακ ισαακ δε εγεννησεν τον ιακωβ ιακωβ δε εγεννησεν τον ιουδαν και τους αδελφους αυτου
ιουδας δε εγεννησεν τον φαρες και τον ζαρα εκ της θαμαρ φαρες δε εγεννησεν τον εσρωμ 
...
```
